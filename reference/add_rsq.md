# Add R-Squares to Endogenous Variables

Replace the residual variances of exogenous variables by their R-squares
in a [qgraph::qgraph](https://rdrr.io/pkg/qgraph/man/qgraph.html)
object.

## Usage

``` r
add_rsq(
  semPaths_plot,
  object = NULL,
  digits = 2L,
  rsq_string = "R2=",
  ests = NULL
)
```

## Arguments

- semPaths_plot:

  A qgraph object generated by
  [`semPaths`](https://rdrr.io/pkg/semPlot/man/semPaths.html), or a
  similar qgraph object modified by other
  [semptools](https://sfcheung.github.io/semptools/reference/semptools-package.md)
  functions.

- object:

  The object used by semPaths to generate the plot. Use the same
  argument name used in
  [`semPaths`](https://rdrr.io/pkg/semPlot/man/semPaths.html) to make
  the meaning of this argument obvious. Currently only object of class
  `lavaan` is supported.

- digits:

  Integer indicating number of decimal places for the R-squares. Default
  is 2L.

- rsq_string:

  The string before the R-squares. Default is `"R2="`.

- ests:

  A data.frame from the
  [`parameterEstimates`](https://rdrr.io/pkg/lavaan/man/parameterEstimates.html)
  function, or from other function with these columns:? `lhs`, `op`,
  `rhs`, and `est`. The rows with `op` equal to `r2` are used to find
  the R-squares. Only used when `object` is not specified.

## Value

If the input is a
[qgraph::qgraph](https://rdrr.io/pkg/qgraph/man/qgraph.html) object, the
function returns a qgraph based on the original one, with R-squares
added. If the input is a list of qgraph objects, the function returns a
list of the same length.

## Details

Modify a [qgraph::qgraph](https://rdrr.io/pkg/qgraph/man/qgraph.html)
object generated by
[`semPaths`](https://rdrr.io/pkg/semPlot/man/semPaths.html) by setting
the labels of the residuals of endogenous variables to their R-squares.

Require either the original object used in the semPaths call, or a data
frame with the R-square for each endogenous variable.

Currently supports only plots based on
[`lavaan`](https://rdrr.io/pkg/lavaan/man/lavaan.html) output.
