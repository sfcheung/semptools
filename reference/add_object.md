# Add a Fit Object to a 'qgraph' Object

Add a fit object (e.g., 'lavaan' output) to the a 'qgraph' object as an
attribute.

## Usage

``` r
add_object(semPaths_plot, object)
```

## Arguments

- semPaths_plot:

  A [qgraph::qgraph](https://rdrr.io/pkg/qgraph/man/qgraph.html) object
  generated by
  [`semPlot::semPaths()`](https://rdrr.io/pkg/semPlot/man/semPaths.html),
  or a similar
  [qgraph::qgraph](https://rdrr.io/pkg/qgraph/man/qgraph.html) object
  modified by other
  [semptools](https://sfcheung.github.io/semptools/reference/semptools-package.md)
  functions.

- object:

  Should be the object, such as the output of
  [`lavaan::sem()`](https://rdrr.io/pkg/lavaan/man/sem.html) or
  [`lavaan::cfa()`](https://rdrr.io/pkg/lavaan/man/cfa.html), used by
  [`semPlot::semPaths()`](https://rdrr.io/pkg/semPlot/man/semPaths.html)
  to generate `semPaths_plot`. Note that this function will not check
  whether the object is appropriate because there is no way to do so
  reliably.

## Value

The original
[qgraph::qgraph](https://rdrr.io/pkg/qgraph/man/qgraph.html) object set
to `semPaths_plot`, with the attribute `"semptools_fit_object"` set to
`object`.

## Details

It adds an object to a
[qgraph::qgraph](https://rdrr.io/pkg/qgraph/man/qgraph.html) object as
the attribute `"semptools_fit_object"`, to be retrieved by other
functions that need to access the original output used in
[`semPlot::semPaths()`](https://rdrr.io/pkg/semPlot/man/semPaths.html)
to create a diagram.

## See also

[`semPlot::semPaths()`](https://rdrr.io/pkg/semPlot/man/semPaths.html)

## Examples

``` r

library(lavaan)
#> This is lavaan 0.6-21
#> lavaan is FREE software! Please report any bugs.
library(semPlot)

mod <-
  'f1 =~ x01 + x02 + x03 + x06
   f2 =~ x04 + x05 + x06 + x07
   f3 =~ x08 + x09 + x10 + x03
   f4 =~ x11 + x12 + x13 + x14
  '
fit <- lavaan::cfa(mod, cfa_example)
p <- semPaths(fit,
              whatLabels = "est",
              sizeMan = 3.25,
              node.width = 1,
              edge.label.cex = .75,
              mar = c(10, 5, 10, 5),
              DoNotPlot = TRUE)
p <- add_object(p, fit)
attr(p, "semptools_fit_object")
```
